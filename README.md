# Event Category Classification

This project is a machine learning model that classifies events into categories such as:
- Technical
- Cultural
- Sports
- Devotional

## Features
- Synthetic dataset generation
- Text preprocessing
- ML model training
- Prediction support

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python main.py
